# Flask Chat Room App

A simple real-time chat room web application built with **Flask** where users can register, log in, and join chat rooms to communicate instantly.

---

## Features

- User Registration and Login (with password hashing)
- Create or join chat rooms
- Real-time messaging using **Flask-SocketIO**
- Simple and responsive UI
- Store users and messages in **SQLite**

---

## Technologies Used

- Python 3.x
- Flask
- Flask-SQLAlchemy
- Flask-SocketIO
- Werkzeug (for password hashing)
- HTML, CSS, Bootstrap (for frontend)
